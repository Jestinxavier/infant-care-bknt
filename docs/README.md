## üöÄ Getting Started

1. **Setup the project:**
   ```bash
   npm install
   cp .env.example .env
   # Edit .env with your credentials
   npm run dev
   ```

2. **Access documentation:**
   - Open browser: http://localhost:3000/api-docs
   - Or read: [Main README](../readme.md)

3. **Test an endpoint:**
   - Visit Swagger UI
   - Click "Try it out"
   - Execute and see response!
